# linëd

/ˈlaɪn.ɛd/

A simple line editing library for Zig.

> [!NOTE]
>
> This is alpha software, and is under development. You should not use it in production.
> The API is subject to (possibly breaking) change.

## Known Issues
- Strange behavior when text wraps.
- You can type escape sequences manually by pressing `ESC` and writing them out.
- No UTF-8 support. (`λ` doesn't work `:(`)
